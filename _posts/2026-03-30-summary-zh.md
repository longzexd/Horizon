3 月 29 日，企业微信在 GitHub 上开源了其 CLI 项目，采用 MIT 许可证。该工具开放了消息、日程、文档、会议等核心业务能力的 API 接口，并支持被主流 AI Agent 调用。 此举大幅降低了开发者和 AI Agent 与企业微信丰富的企业协作工具集成的门槛。它使得自动化复杂业务流程成为可能，并可能加速企业级 AI 助手的开发，这些助手可以直接在公司的通信平台内管理日程、发送消息或创建文档。 该 CLI 工具覆盖了 7 大业务品类，并提供了 12 个具体的 AI Agent Skills。它可以通过 npm 安装，在终端完成配置后即可调用相关能力。

## [工程化精液外泌体实现非侵入性眼药水疗法，用于治疗视网膜癌](https://t.me/zaihuapd/40599) ⭐️ 7.0/10

研究人员开发了一种新型眼药水制剂，利用工程化的精液来源外泌体（SEVs）为视网膜母细胞瘤等视网膜疾病提供靶向治疗。这些 SEVs 通过表面修饰叶酸（FA）实现靶向，并装载了 CMG 纳米酶系统以杀死癌细胞；由于其表面表达的表皮生长因子（EGF），它们能有效穿透眼部屏障。 这项研究为治疗眼底疾病提供了潜在的突破，为侵入性注射或手术提供了一种非侵入性的替代方案。如果成功，它可能通过提高靶向性和减少全身副作用，彻底改变眼部疾病的药物递送方式，特别是对于视网膜母细胞瘤等侵袭性癌症。 该研究目前处于动物实验阶段，要确保其用于人类的安全性和长期疗效，仍需大量研究。其关键机制在于 SEV 表面的 EGF 能暂时打开角膜和结膜的紧密连接，从而使载药囊泡能够到达眼底（眼球后部）。

**背景**: 外泌体是天然存在的、细胞衍生的纳米囊泡，在细胞间运输物质，由于其生物相容性和穿越生物屏障的能力，正被探索作为药物递送载体。视网膜母细胞瘤是一种罕见但侵袭性强的眼癌，主要影响幼儿，目前的治疗方法可能具有侵入性，并存在视力丧失或全身毒性的风险。工程化外泌体涉及修饰其表面或内部载荷，以增强对特定疾病的靶向性和治疗效果。
