rss · Bloomberg Markets · Apr 21, 08:28

**背景**: 霍尔木兹海峡是位于伊朗和阿曼之间的狭窄水道，连接波斯湾与阿曼湾及公海。它是全球最重要的石油运输咽喉要道，每日有大量全球贸易的原油、凝析油和液化天然气通过该海峡。该海峡周围的地缘政治紧张局势历来会导致油价波动，因为市场需要评估供应中断的风险。

## [Claude Desktop 静默安装浏览器集成文件，未经用户同意](https://www.thatprivacyguy.com/blog/anthropic-spyware) ⭐️ 7.0/10

Anthropic 的 Claude Desktop 应用在 macOS 安装后，会自动在七个 Chromium 浏览器目录中创建原生消息传递主机清单文件。该文件名为 com.anthropic.claude_browser_extension.json，预授权三个特定的 Chrome 扩展 ID 与本地二进制文件通信，从而启用浏览器标签打开、登录状态共享和 DOM 读取等功能。 这一行为构成了重大的隐私和安全问题，因为它在用户不知情或未同意的情况下，在浏览器和桌面应用之间建立了通信桥梁。对于 Anthropic 这样的主要 AI 公司，这种未记录的集成可能会将用户的浏览数据和认证会话暴露给应用程序，为软件透明度开创了一个令人不安的先例。 安装发生在 Claude Desktop 应用启动时，且清单文件会在后续每次运行时被重写，甚至为未安装的浏览器也创建了目录。Anthropic 的公开文档未提及此桌面桥接功能，仅单独记录了 Claude Code 的集成说明。

**背景**: 原生消息传递（Native Messaging）是一项浏览器功能，允许扩展程序与用户计算机上安装的本地应用程序进行通信。其工作原理是注册一个清单文件，指定一个主机应用程序，浏览器随后可以启动该程序来交换 JSON 消息。此机制通常用于合法的集成，例如密码管理器与桌面应用通信，但需要用户明确同意才能安装扩展。NativeMessagingHosts 文件夹是浏览器查找这些注册文件的标准位置。
