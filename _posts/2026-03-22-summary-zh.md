## [开发者演示通过公开评论历史对 Hacker News 用户进行 AI 画像分析](https://simonwillison.net/2026/Mar/21/profiling-hacker-news-users/#atom-everything) ⭐️ 7.0/10

2026 年 3 月 21 日，开发者 Simon Willison 发布了一项技术探索，展示了如何使用 Algolia Hacker News API 获取用户最近的 1000 条评论，并将其输入到 Claude Opus 4.6 等大语言模型中，并附上“分析此用户”的指令。AI 仅基于公开的评论数据，就成功生成了详细的个人和专业画像，包括技术兴趣、工作风格甚至安全关切。 这表明，易于获取的公开数据与强大的 AI 分析相结合，可以被用来推断敏感的个人和专业信息，引发了重大的隐私和安全担忧。它凸显了对在线社区中的个人进行自动化、大规模行为画像分析的潜力，这可能被用于定向广告、社会工程学或其他侵入性目的。 Algolia API 提供了开放的 CORS 头，允许来自任何网页的 JavaScript 直接访问而无需认证，使得数据收集变得轻而易举。Claude 生成的 Willison 个人画像准确地识别出他是 Django 的联合创始人、他对“智能体工程”的倡导、他的工作习惯，甚至是他对提示注入的具体安全关切。
